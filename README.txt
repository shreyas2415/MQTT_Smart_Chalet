MQTT based Project for Smart Chalet
MQTT based protocol for SMART CHALET

This is a MQTT based protocol to talk to the Photon to do crazy stuffs. 

If you send the following Messages, you would see the following things to happen on the Photon
RED - Sets RGB LED as Red
GREEN - Sets RGB LED as Green 
BLUE - Sets RGB LED as Blue
STOP - Sets RGB LED White
SWITCH_D7 - Blinks the LED D7 5 times 
No message - All LEDs OFF
